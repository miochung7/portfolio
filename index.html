about me

I’ve been teaching myself since 2019. I am looking for my first software development in front-end (aspiring full-stack) Web Developer.

When a friend of mine suggested programming to me, I immediately knew it was the direction I wanted to go, and I haven’t looked back since. Attending regular workshops, meet-up events with Code First Girls, She Can Code, CodeBar and You Got This etc continously inspires me how our ideas can be brought to life and also motivates me to build projects using the latest technologies.

Current stack:
HTML5, CSS3, Sass, JavaScript ES6+, Node.js, Express.
